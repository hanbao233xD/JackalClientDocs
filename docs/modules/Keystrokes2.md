介绍
Keystrokes2（按键显示II）用于暂无描述。
适合键鼠行为控制、输入增强和自动化操作场景。
初次使用可优先调整：Toggle KeyReaction Module、Rain Effect Coordinate Mode、Multi Row Mode。

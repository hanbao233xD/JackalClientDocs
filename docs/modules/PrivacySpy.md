PrivacySpy
隐私密探
分类：File
描述：根据本地文件汇总用户的隐私信息，让你变得紧张。

需求
- 安全级别：高风险模块
- 恶意标记：是（malware）
- 权限需求：无
- 驱动依赖：否
- 联网需求：否
- 开发状态：稳定
- 版本属性：普通可用

介绍
PrivacySpy 会扫描指定目录的文件名与目录名，提取可能的人名、身份证号、手机号，并输出统计与可疑路径列表。
它还支持“其他隐私线索”检测（例如常见软件路径痕迹），用于快速评估本机可被侧推的隐私暴露面。模块不需要联网，但结果高度敏感，建议仅在授权环境下使用。

配置项
- Async（异步）
  类型：布尔；默认：true；说明：开启后异步扫描，主界面可继续操作；目录较大时建议开启，避免界面长时间等待。
- Auto Console（自动启用控制台）
  类型：布尔；默认：true；说明：开启后会自动打开或聚焦控制台，扫描统计与可疑路径都在控制台输出；建议开启便于完整查看结果。
- Custom Scan Directory（自定义扫描目录）
  类型：文本；默认："D:\\"；说明：指定扫描根目录。路径不存在会直接终止任务；建议从较小目录开始逐步扩大，先验证规则再全盘扫描。
- Skip Folders（跳过目录检查）
  类型：布尔；默认：false；说明：关闭时会同时检查目录名和文件名；开启后仅看文件名。若你只关注文件泄露，可开启以降低噪声。
- Skip Readonly Files（跳过只读文件）
  类型：布尔；默认：true；说明：开启后忽略只读属性目标，减少系统文件与程序文件干扰；做全量审计时可临时关闭。
- Skip Hidden Files（跳过隐藏文件）
  类型：布尔；默认：true；说明：开启后跳过隐藏文件，结果更偏向用户可见内容；若要做深度排查可关闭。
- Skip System Files（跳过系统文件）
  类型：布尔；默认：true；说明：开启后跳过系统文件，降低误报和扫描量；定位系统侧泄露线索时可关闭。
- IDCard Valid Year Min（身份证有效年份最小值）
  类型：整数；默认：1914；说明：用于身份证出生年份下限过滤。范围过低会增大误报，建议与目标人群年龄段一致。
- IDCard Valid Year Max（身份证有效年份最大值）
  类型：整数；默认：2024；说明：用于身份证出生年份上限过滤。建议按当前年份更新，避免未来年份导致误判。
- IDCard Strict Validify（身份证严格验证）
  类型：布尔；默认：true；说明：开启后会额外校验前导数字边界、出生日期合法性与校验位，误报更低；建议保持开启。
- Person Name Character Max Count（人名最多几个汉字）
  类型：整数；默认：3；说明：限制姓名最大字数。默认更贴近日常姓名，改大后会捕获更多长词但噪声也会上升。
- Filter File Extensions（筛选文件扩展名）
  类型：布尔；默认：true；说明：开启后只扫描 Included File Extensions 中的扩展名，显著降低扫描量与误报；建议保持开启。
- Included File Extensions（包括的文件扩展名）
  类型：文本；默认：长扩展名列表；说明：分号分隔扩展名白名单。建议按业务文档类型补充，不要盲目加入全部二进制格式。
- Scan Disk Privacy（扫描磁盘隐私）
  类型：布尔；默认：true；说明：控制是否进入磁盘扫描流程。关闭可仅使用“其他隐私线索”检测。
- Confirm Before Scanning（扫描前确认）
  类型：布尔；默认：true；说明：开启后每次扫描前弹确认框，防止误触发大范围扫描；建议保留。
- Find Other Privacy（寻找其他隐私）
  类型：布尔；默认：true；说明：开启后先检测一组预置隐私线索路径并输出命中结果，适合快速预判风险面。

备注
PrivacySpy 依赖 `assets/PERSON_SURNAMES.TXT` 与 `assets/PERSON_EXCLUDE.TXT` 辅助识别。若你发现人名识别偏差，优先调整这两个词表与扩展名白名单。

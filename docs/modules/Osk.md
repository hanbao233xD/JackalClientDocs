Osk
屏幕键盘
分类：Process
描述：打开屏幕键盘。

需求
- 安全级别：常规模块
- 恶意标记：否
- 权限需求：无
- 驱动依赖：否
- 联网需求：否
- 开发状态：稳定
- 版本属性：普通可用

介绍
Osk 用于打开系统屏幕键盘（`Osk.exe`）。
关闭模块时会先尝试关闭屏幕键盘窗口，再回退到进程结束，减少残留实例。

配置项
无（该模块在 default_settings.h 中未定义独立配置项）

历史更新
无（HISTORY 中暂无明确记录）

备注
模块状态与 `osk.exe` 进程状态联动：手动关闭屏幕键盘后模块会自动关闭；手动打开时模块会自动显示为开启。

AntiProcCreate
禁止创建进程
分类：未分类
描述：使用驱动程序阻止新进程的创建。

需求
- 安全级别：常规模块
- 权限需求：无
- 驱动依赖：是
- 联网需求：否
- 开发状态：稳定/常规
- 版本属性：普通可用

介绍
AntiProcCreate（禁止创建进程）用于使用驱动程序阻止新进程的创建。
适合在日常管理与自动化场景中按需启用。

配置项
无（该模块在 default_settings.h 中未定义独立配置项）

历史更新
- 16. 添加使用驱动的模块： AntiProcCreate，启用后将因为打印机缺纸（即使你没有打印机，但是就是缺纸），阻止进程创建。

备注
该模块可能受系统版本、权限级别、目标进程状态或安全软件策略影响；若功能未生效，优先检查管理员权限、驱动依赖、联网状态与系统兼容性。

介绍
FolderClear（目录清空）用于暂无描述。
适合在日常管理与自动化场景中按需启用。
初次使用可优先调整：Filter、Notify Errors、Notify Result。

- Async（中文：异步）：类型=布尔，默认=true
- Filter（中文：筛选器）：类型=文本，默认="*.*"
- Notify Errors（中文：通知错误）：类型=布尔，默认=true
- Notify Result（中文：通知结果）：类型=布尔，默认=true

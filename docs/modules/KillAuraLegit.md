KillAuraLegit
合法杀戮光环
分类：未分类
描述：自动移动鼠标到目标窗口右上角然后关闭它。

需求
- 安全级别：常规模块
- 恶意属性：否
- 权限需求：无
- 驱动依赖：否
- 联网需求：否
- 开发状态：稳定/常规
- 版本属性：普通可用

介绍
KillAuraLegit 会先把鼠标逐步移动到目标窗口的关闭按钮附近，再执行关闭动作，整体行为比 KillAura 更接近“人工点关闭”。
模块会优先确保目标处于前台或鼠标已在目标窗口上；当鼠标足够接近关闭点时，按节流间隔触发一次关闭。

配置项
- Smoothness（平滑度）
  类型：数值；默认：0.8f。用于控制“靠近关闭按钮”的移动速度：值越高越平滑、越慢；值越低越快、越生硬。默认值更偏自然轨迹，若希望更快执行可降低到 0.3~0.6。

历史更新
无（HISTORY 中暂无明确记录）

备注
该模块虽然不在 unsafe 列表中，但本质仍是自动关窗动作，建议先在非关键窗口验证手感后再投入日常使用。
如果你不需要“先移动再关闭”的拟人过程，而是追求更直接的持续关窗，可改用 KillAura。

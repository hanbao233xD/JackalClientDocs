RecyclerClear
回收站清空
分类：File
描述：清空系统回收站。

需求
- 安全级别：常规模块
- 恶意标记：否
- 权限需求：无
- 驱动依赖：否
- 联网需求：否
- 开发状态：稳定
- 版本属性：普通可用

介绍
RecyclerClear 调用系统回收站接口执行清空，而不是逐文件删除。它支持声音、确认弹窗、进度条三类系统 UI 开关，适合做“最后一步”统一清空。
模块内部带并发保护，已有任务运行时会拒绝重复触发，避免多次并行清空造成冲突。

配置项
- Async（异步）
  类型：布尔；默认：true；说明：开启后后台执行并立即返回“请求已发送”提示；关闭后会等待系统清空流程执行完成再返回。
- Sound（音效）
  类型：布尔；默认：true；说明：控制系统回收站清空完成音效；批处理场景建议关闭，避免频繁提示音。
- Confirm（确认）
  类型：布尔；默认：true；说明：控制是否显示系统确认框；对误操作敏感的环境建议保持开启。
- Progress Bar（进度条）
  类型：布尔；默认：true；说明：控制是否显示系统进度界面；自动化执行可关闭以减少界面干扰。

备注
清空失败时会返回系统 HRESULT 错误。若长期失败，优先检查回收站状态、资源管理器状态和权限环境。

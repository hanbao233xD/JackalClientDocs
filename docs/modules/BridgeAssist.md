介绍
BridgeAssist（搭路辅助）用于暂无描述。
适合在日常管理与自动化场景中按需启用。
初次使用可优先调整：Mode。

配置项
- Check Window（检查窗口）
 类型：布尔；默认：true
 说明：用于指定模块实际作用对象。建议先对单个目标测试通过，再扩大到多目标，降低误操作风险。
- Trigger Condition（触发条件）
 类型：枚举；默认："S & D"
 说明：这是选项型配置。默认值 S & D 一般更稳妥；建议按使用场景逐个试用，而不是一次性切换多项。
 可选：S（未收录）；S & A（未收录）；S & D（未收录）
- BPS（每秒方块数。这个控制操作的频率）
 类型：数值；默认：2.10f
 说明：这是数值型配置。默认值 2.10f 通常在稳定性与效果之间做了平衡，建议小步调整并观察实际变化。
- Mode（模式）
 类型：枚举；默认："Sneak"
 说明：这是该模块的核心行为开关，不同选项对应不同执行策略。建议先保持默认 Sneak ，确认稳定后再逐项切换比较效果。
 可选：Simple（简易）；Sneak（潜行）
- Sneak Key（潜行键）
 类型：按键/复合；默认：`{ { "Keybind", {VK_LSHIFT } }`}
 说明：该配置用于调整模块行为细节。建议先按默认值运行，确认需求后再逐步调整。
- Place Key（放置键）
 类型：按键/复合；默认：`{ { "Keybind", {VK_RBUTTON } }`}
 说明：该配置用于调整模块行为细节。建议先按默认值运行，确认需求后再逐步调整。
- Sneak Reshift Gap（潜行后重新按Shift间隔）
 类型：数值；默认：50U
 说明：这是数值型配置。默认值 50U 通常在稳定性与效果之间做了平衡，建议小步调整并观察实际变化。
- Right Internal Gap（右键内部间隔）
 类型：数值；默认：10U
 说明：这是数值型配置。默认值 10U 通常在稳定性与效果之间做了平衡，建议小步调整并观察实际变化。
- Shift Internal Gap（Shift键内部间隔）
 类型：数值；默认：100U
 说明：这是数值型配置。默认值 100U 通常在稳定性与效果之间做了平衡，建议小步调整并观察实际变化。
- Place Sound（放置音效）
 类型：布尔；默认：true
 说明：用于选择结果反馈方式。默认值 true 适合大多数场景；若你不想打扰可改为更安静的输出方式。

AimAssist
自动瞄准
分类：未分类
描述：自动瞄准目标窗口的右上角。

需求
- 安全级别：常规模块
- 恶意属性：否
- 权限需求：无
- 驱动依赖：否
- 联网需求：否
- 开发状态：稳定/常规
- 版本属性：普通可用

介绍
AimAssist 会自动把鼠标往目标窗口右上角（接近关闭按钮的位置）推进，让“指向目标并关闭”这类操作更省力。
它不是瞬移，而是按平滑系数逐步移动；因此既可以做快速吸附，也可以做更自然的缓动跟随。

配置项
- Smoothness（平滑度）
  类型：数值；默认：0.8f。该值直接控制鼠标每帧朝目标点靠近的比例：值越高，单次位移越小、移动更柔和；值越低，吸附更快更“硬”。若你想快速锁到关闭按钮可降到 0.2~0.5，若要更像人工移动可保持 0.7~0.9。

历史更新
无（HISTORY 中暂无明确记录）

备注
AimAssist 会持续改写鼠标位置，不适合与需要精细手动拖拽的操作同时开启。
若你的目标是“自动关窗”而非单纯辅助瞄准，可与 TriggerBot 联动，或直接改用 KillAuraLegit。

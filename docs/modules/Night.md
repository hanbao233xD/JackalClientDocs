介绍
Night（夜间）用于暂无描述。
适合在日常管理与自动化场景中按需启用。
初次使用可优先调整：Hide When No Focus、Mask Alpha Rate、When。

配置项
- Hide When No Focus（中文：无焦点时隐藏）：类型=布尔，默认=true
- Mask Alpha Rate（中文：遮罩不透明度比例）：类型=数值，默认=0.8f
- When（中文：何时）：类型=枚举，默认="Menu On"
  可选：Menu On（打开菜单）；Menu Off（关闭菜单）；Always（总是）

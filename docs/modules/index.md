# 模块介绍

JackalClient 采用模块化设计，每个功能都以独立模块形式存在，可按需启用。

## 模块文档入口

你可以直接浏览 `docs/modules` 下的单模块说明文档，了解：

- 模块用途与适用场景
- 基础命令与配置方式
- 使用注意事项

## 推荐阅读顺序

1. 先阅读《功能与模块地图》把握全局能力：[/module-map](/module-map)
2. 再进入本目录按需查看具体模块：[/modules/](/modules/)
3. 配合《完整命令手册》进行命令级检索：[/commands/](/commands/)

## 典型模块分类

- 系统与窗口控制
- 进程与安全相关
- 自动化与快捷操作
- 在线服务与内容增强

> 提示：如果你是首次使用，建议先看《快速开始》再进入模块文档。

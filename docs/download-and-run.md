::: danger ！！！警示！！！
如果你需要达成“下载并运行 JackalClient 软件”的任务，你需要完成“阅读网页”的操作。请不要感到害怕或灰心，按照指示一步步操作，就可以成功。
:::

1.  **必要器官**：

## 🚀 详细操作步骤

### 1. 启动 QQ
1.  在你的计算机屏幕（显示器）上，找到 **QQ** 的图标（企鹅图案）。
2.  握住 **鼠标**，移动光标到图标上方。
3.  **双击** 鼠标左键打开 QQ。
4.  等待 QQ 主窗口出现。

### 2. 进入目标群聊
1.  在 QQ 面板的左侧或中间找到 **消息列表**。
2.  使用鼠标滚轮滚动，找到包含 JackalClient 文件的群聊。
3.  **单击** 群名称，进入聊天界面。

### 3. 打开群文件
1.  在群聊天窗口顶部或侧边的功能栏中，找到 **“文件”** 按钮（或文件夹图标）。
2.  **单击** 进入群文件界面。

### 4. 找到并下载压缩包
1.  在文件列表中找到名为 **JackalClient** 的文件夹，**双击** 进入。
2.  找到最新版本的压缩包，名称类似：`JackalClient_v0.8d_Pwd：Wormwake.zip`。
    *   **JackalClient**：软件名称。
    *   **v0.8d**：版本号。
    *   **Pwd：Wormwake**：解压密码提示。
    *   **.zip**：压缩包格式。
3.  **右键单击** 该文件，选择 **“下载”**。
4.  等待下载完成。默认通常保存在 `C:\Users\你的用户名\Downloads`（下载文件夹）。

### 5. 找到下载的文件
1.  在任务栏底部找到 **文件夹图标**（文件资源管理器），**单击** 打开。
2.  在左侧导航栏找到并点击 **“下载”** 或 **“Downloads”**。
3.  在右侧列表中找到刚才下载的 `JackalClient_v0.8d_Pwd：Wormwake.zip`。

### 6. 正确解压文件（关键步骤）

::: warning ⚠️ 重要提醒
**绝对不要** 直接双击打开压缩包后直接运行里面的程序！这会导致“找不到文件”等错误。
**必须** 使用压缩软件进行“完整解压”。
:::

1.  **右键单击** 压缩包文件。
2.  选择压缩软件选项（如 **WinRAR**、**7-Zip** 或 **Bandizip**）。
    *   *如果没有这些选项，请先安装一个压缩软件。*
3.  选择 **“解压到...”** 或 **“提取到...”**。
4.  选择一个你容易找到的位置（例如 **桌面** 或 **D盘**）。
    *   建议新建一个名为 `JackalClient` 的文件夹。
5.  在弹出的密码输入框中，输入解压密码：
    > **Wormwake**
    
    *(注意区分大小写，W 大写，不要有多余空格)*
6.  点击 **“确定”** 开始解压。

### 7. 处理杀毒软件误报
解压后，Windows Defender 或其他杀毒软件可能会误报并拦截文件。

1.  如果提示“发现威胁”，点击任务栏的安全中心图标。
2.  进入 **“病毒和威胁防护”** -> **“保护历史记录”**。
3.  找到与 JackalClient 相关的拦截记录，选择 **“允许在设备上”** 或 **“还原”**。
4.  **添加信任**：
    *   进入“病毒和威胁防护”设置 -> “管理设置” -> “排除项”。
    *   点击“添加排除项” -> “文件夹”。
    *   选择你刚才解压出来的 `JackalClient` 文件夹。

### 8. 运行程序
1.  进入解压后的 `JackalClient` 文件夹。
2.  找到 **`Launcher.exe`**（启动器）。
3.  **双击** 运行。
    *   如果弹出用户账户控制（UAC）提示，选择 **“是”**。
4.  如果启动器无法运行，可以尝试进入 `Release` 子文件夹，直接运行 `JackalClient.exe`。
